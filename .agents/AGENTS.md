# Project Rules

## Bibliography Formatting
When adding or modifying features that generate bibliographies or citation references, you MUST follow these specific formatting rules for various citation styles based on the number of authors (1, 2, or 3+) and source type (Book vs Journal).

### APA Style (7th Edition)
- **Book (1 Author)**: Lastname, Initial. (Year). *Book Title*. Publisher.
- **Book (2 Authors)**: Lastname 1, Initial., & Lastname 2, Initial. (Year). *Book Title*. Publisher.
- **Book (3+ Authors)**: Lastname 1, In., Lastname 2, In., & Lastname 3, In. (Year). *Book Title*. Publisher.
- **Journal**: Lastname, Initial. (Year). Article title. *Journal Name*, Volume(Issue), Pages. DOI/URL

### MLA Style (9th Edition)
- **Book (1 Author)**: Lastname, Firstname. *Book Title*. Publisher, Year.
- **Book (2 Authors)**: Lastname 1, Firstname 1, and Firstname 2 Lastname 2. *Book Title*. Publisher, Year.
- **Journal**: Lastname, Firstname. "Article Title." *Journal Name*, vol. X, no. Y, Year, pp. A-B.

### Harvard Style
- **Book (1 Author)**: Lastname, Initial. (Year) *Book Title*. City: Publisher.
- **Book (2 Authors)**: Lastname 1, Initial. and Lastname 2, Initial. (Year) *Book Title*. City: Publisher.
- **Journal**: Lastname, Initial. (Year) 'Article title', *Journal Name*, Vol(Issue), pp. A-B.

### Chicago Style (Author-Date)
- **Book (1 Author)**: Lastname, Firstname. Year. *Book Title*. City: Publisher.
- **Book (2 Authors)**: Lastname 1, Firstname 1, and Firstname 2 Lastname 2. Year. *Book Title*. City: Publisher.
- **Journal**: Lastname, Firstname. Year. "Article Title." *Journal Name* Vol, no. Issue: A-B.

### IEEE Style
- **Book**: [Number] First Initial. Lastname, *Book Title*, City: Publisher, Year.
- **Journal**: [Number] First Initial. Lastname, "Article title," *Journal Name*, vol. X, no. Y, pp. A-B, Month Year.

### Vancouver Style
- **Book**: Number. Lastname Initial. Book Title. City: Publisher; Year.
- **Journal**: Number. Lastname Initial. Article title. Journal Abbreviation. Year;Vol(Issue):A-B.

*Tip: Pay attention to name inversion (only first author vs all authors) and formatting (italics vs quotes).*
